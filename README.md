# CreditLens | Bank Loan Analytics Dashboard

## Overview

CreditLens is an end-to-end Banking Loan Analytics project designed to analyze loan performance, borrower behavior, funding trends, and portfolio risk using SQL and Power BI.

The project transforms raw banking loan data into actionable business insights through KPI tracking, data modeling, DAX calculations, and interactive dashboards.

---

## Project Objectives

* Analyze loan application trends
* Monitor funded vs received amounts
* Evaluate portfolio performance
* Identify Good Loans and Bad Loans
* Measure loan repayment efficiency
* Track borrower demographics and loan purposes
* Support data-driven lending decisions

## Key Insights

* Processed 38K+ banking loan applications.
* Total funded amount exceeded $435 Million.
* Loan repayments generated over $473 Million.
* Average portfolio interest rate remained around 12%.
* Good loans dominated the portfolio.
* Charged-off loans highlighted credit risk exposure.
* Debt-to-Income ratio significantly influenced loan quality.
* Geographic trends revealed lending concentration across multiple states.
